# Dynamic Quiz Application with Selenium Automation

## Project Overview
This project is a dynamic quiz application developed using HTML, CSS, and JavaScript. 
Users can select a difficulty level, answer timed questions, and view result analysis 
after completing the quiz.

## Features
- Difficulty-based quiz (Easy, Medium, Hard)
- Countdown timer for each question
- Automatic question navigation
- Result analysis with correct and incorrect answers
- Responsive UI

## Automation Testing
Selenium WebDriver (Java) is used to automate the quiz workflow:
- Launch the quiz application
- Select difficulty level
- Start quiz
- Select answers for each question
- Navigate through questions
- Verify result screen

## Tools & Technologies
- HTML, CSS, JavaScript
- Java
- Selenium WebDriver
- VS Code
- GitHub

## How to Run
1. Open `index.html` in a browser to run the quiz manually
2. Run `QuizAutomationTest.java` to execute Selenium automation
